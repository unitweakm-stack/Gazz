# Telegram Music Bot 🎵

Telegram bot to publish music with images to a channel

## Features
- Upload music (MP3) + image together
- Automatically posts to Telegram channel
- Shows song info on the image caption

## Deploy on Railway

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/your-template)

### Environment Variables
- `BOT_TOKEN`: Your Telegram bot token
- `CHANNEL_ID`: Target channel ID (e.g., -1001234567890)

### Setup
1. Create bot via @BotFather
2. Add bot as admin to your channel
3. Set environment variables on Railway
4. Deploy!

## Usage
1. Send /start to bot
2. Send MP3 + image together in one message
3. Bot posts to channel automatically
